### Rogue-OOP
Turn based 'Rogue-Lite' built entirely using C++ and the SFML graphics library.


Made for OOP Semester 2 - 2022
- Character sprites made externally by Eli Cuaycong

### Build:
- SFML Required
- make main ( for game )
- make dSprite, sprite, text, entityStats ( for tests )
